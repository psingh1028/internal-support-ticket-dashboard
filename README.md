# Internal Support Ticket Dashboard

An ASP.NET Core Web API project for managing internal support tickets.

## Current Features

- Get all tickets
- Get a ticket by ID
- Create a new ticket
- Update an existing ticket
- Delete a ticket
- Test API endpoints using Swagger

## Tech Stack

- C#
- ASP.NET Core Web API
- Swagger / OpenAPI
- Git / GitHub

## Project Goal

This project is being built to practice and demonstrate backend API development, CRUD operations, and future SQL Server integration.
